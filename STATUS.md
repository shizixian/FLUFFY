## 当前状态（2026-06-13）
- **难度提升**：关卡成长更快（target 16 +5/level），类型数量更早增加（4+level），布局中心堆叠加深到 layer3（更多牌被埋，表面可消对更少，更 puzzle 化）。生成器尝试次数增至120，保持可解性但显著变难。早期关卡从16张5种开始。
- **可玩原型**：已实现基础消除逻辑，能玩到通关。
- **布局**：固定 2D 布局（gx/gy + layer 深度），**洗牌不会导致“行数/整体跨度”变少**。布局跨度固定，洗牌时剩余牌会随机打散到整个原始布局区域。
- **选中规则**（已符合你要求）：
  - 只能选中“free”的牌：没被更高 layer 盖住 + 同一 layer 左右至少一边是空的（可以“滑出来”）。
  - “羊后面如果还是羊”一样可以选（只要你点的这张侧边有空位就行）。
- **视觉**：纯图片风格卡牌（彩色卡底 + 中心卡通图案图标），**完全没有文字**。
  - 现在每个方块使用独立的卡通图案图片（扁平可爱风格：羊头、草丛、米穗、田地、兔子、狼头）。
  - 图片放在 `assets/icons/`，通过 TextureRect 加载（保留了原来的纸牌背景和边框）。
  - 你可以随时用网上找到的更好 PNG 素材替换这些 .jpg 文件（搜索“羊了个羊 素材 png”或免费图标站）。
- **核心机制**：
  - 点两张 free + 类型相同的牌即可消除。
  - 消除后自动检查是否还有解。
  - 洗牌按钮：收集剩余牌类型，**优先使用可解重排生成器**（对剩余牌应用反向放置，保证剩余也能完整消除），必要时 fallback 随机 + 少量重试。
  - 卡住时会提示：“牌面已无解！点击【洗牌】继续”。
  - 关卡系统、得分、剩余数量显示。
  - 通关条件：剩余牌 == 0 时弹出通关界面并加分。
- **完美可解布局（本次改进核心）**：
  - 初始发牌使用 **反向配对放置（reverse pair placement）**：从空板开始，依次在“当前可放置（placeable）”的位置上成对放入相同类型。
  - 每种类型数量严格为偶数（pairs_total 计算 + 每类型分配整对），杜绝“孤儿牌”。
  - 结果：关卡开始时就保证存在完整消除序列。你应该能一直匹配到 0，不会出现“因为布局本身导致最后剩牌无法匹配、必须洗牌”的情况。
  - 洗牌保留且强化：优先给剩余牌也生成一个可解布局，让你洗牌后通常也能直接玩完。
  - 注意：如果你操作顺序很“贪心”/不佳，仍可能自己制造阻塞（这是消除类游戏的正常乐趣），这时洗牌就是很好的补救。
- **已解决的主要问题**：
  - 点击卡牌无效（改成手动边界检测 + Camera2D）。
  - 消除后节点被提前 free 导致 highlight 报错（清除选中状态前置 + 保护判断）。
  - 洗牌后布局越来越小 / 行数变少（固定布局 + 随机打散）。
  - 玩到最后剩牌不匹配、过不了关（根因是随机分配 + 可能出现单数类型；现已通过“偶数分配 + 可解生成器”从源头解决。初始布局和洗牌后基本都能完整通关，不再依赖洗牌救场）。
  - 旧的列堆叠模型（只点最上面）已完全替换为 2D 布局。
  - 类型分配可能产生奇数导致的“天生无法消完”（现修复）。
