# 隐私政策

**最后更新日期：2026年6月19日**

FLUFFY羊羊乐（以下简称"本游戏"）由 ShiZixian（以下简称"我们"）开发。本隐私政策说明我们如何收集、使用和保护您的信息。

## 我们收集的信息

**我们不收集任何个人信息。**

本游戏：
- 不需要注册账号
- 不收集位置信息
- 不收集设备信息
- 不收集使用数据
- 不需要网络连接

所有游戏数据（用户名、头像选择、最高关卡、总分）仅保存在您的设备本地，不会上传到任何服务器。

## 第三方服务

本游戏不使用任何第三方分析、广告或追踪服务。

## 儿童隐私

本游戏适合所有年龄段用户。我们不会故意收集任何人的个人信息，包括13岁以下儿童。

## 政策变更

如果我们的隐私政策发生变化，我们会在本页面更新。

## 联系我们

如有任何关于本隐私政策的问题，请通过以下方式联系我们：
- GitHub Issues: https://github.com/shizixian/FLUFFY/issues
